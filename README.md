# Esm

> Built to remeber when the world tries to forget

Esm is built for you to focus on doing :saluting_face:
Just ask it to `todo`!

It's:
- text-based
- easy to learn
- ***Very*** fast
- Structured

All you need to do is:
1. download it from here
2. double click it
3. add your tasks
4. let it manage your tasks for you

Features:
- [X] Managing Tasks
- [X] Manage Events
- [X] Find Tasks
- [ ] Reminders (coming soon)

Some basic commands are
```
todo <task>
deadline <task \by yyyy-mm-dd>
```
